# 🤟 ISL Audio-to-Sign Web Translator
**Bridging the gap between sound and Indian Sign Language (ISL)**

## 🌟 Overview
This project is an automated web-based translator designed to convert spoken Hindi/English audio into Indian Sign Language (ISL) animations in real-time. By leveraging Google Speech-to-Text and MediaPipe, it provides an accessible communication tool for the Deaf and Hard of Hearing community in India.

## ⚖️ Tech & Ethics (The Developer-Lawyer Perspective)
As a Computer Science student and aspiring lawyer, this project was built with a focus on **Digital Accessibility (A11y)** and **Data Privacy**:
* **Privacy by Design:** No audio data is stored locally after processing; it is streamed directly to the API for real-time translation.
* **Compliance:** Developed with an eye toward future digital accessibility standards in India.

## 🛠️ Tech Stack
* **Frontend:** HTML5, CSS3, JavaScript (Vanilla)
* **Speech Engine:** Google Cloud Speech-to-Text API
* **Motion Tracking:** MediaPipe (for skeletal landmark mapping)
* **3D Assets:** Custom rigged avatars created in **Blender**
* **Deployment:** GitHub Pages
